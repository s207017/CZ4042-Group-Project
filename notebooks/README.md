# Notebooks

This directory contains Jupyter notebooks for exploration and experimentation.

## Planned Notebooks

- `01_data_exploration.ipynb`: Explore datasets, distributions, statistics
- `02_preprocessing.ipynb`: Test preprocessing pipelines
- `03_baseline_models.ipynb`: Train and evaluate baseline models
- `04_transformer_experiments.ipynb`: Compare transformer architectures
- `05_domain_adaptation.ipynb`: Domain adaptation experiments

## Usage

To use these notebooks, activate the conda environment:

```bash
conda activate sentiment-analysis
jupyter lab  # or jupyter notebook
```
